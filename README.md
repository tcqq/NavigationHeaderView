# NavigationHeaderView
HeaderView for NavigationView.

# Setup
#### build.gradle
```
repositories {
    maven { url 'https://jitpack.io' }
}
```
```
dependencies {
    // Using JCenter
    implementation 'com.tcqq.android:navigationheaderview:1.0.0'
}
```
